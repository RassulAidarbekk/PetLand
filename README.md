PetLand is a decentralized pet breeding, collecting, and trading dApp built on Sepolia testnet.  
Users can create pets, merge them for hybrids, list them on the market, and buy/sell using PetCoin (ERC-20 token).

## Features

- Create new pets (costs 10 PetCoin)
- Merge two pets into a new hybrid (costs 20 PetCoin)
- List pets for sale on the marketplace
- Buy pets from other users
- View your pets and the last created pet on home page
- Connect with MetaMask (Sepolia network)

## Tech Stack

- **Frontend**: HTML, CSS, JavaScript, ethers.js v5
- **Backend**: Node.js, Express, MongoDB (Atlas)
- **Blockchain**: Sepolia testnet, custom ERC-20 token (PetCoin)
- **Storage**: MongoDB for pet metadata
